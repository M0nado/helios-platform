
**Check Build Status:**
```powershell
# See all available builds
.\scripts\build-manager\list-builds.ps1

# Compare two builds
.\scripts\build-manager\compare-builds.ps1 -Build1 "phase-1" -Build2 "phase-3"

# Enable/disable features
.\scripts\build-manager\toggle-feature.ps1 -Feature "optimization-level-3" -Enable $true
```

**Use AI Services:**
```powershell
# Get ChatGPT suggestions for build optimization
.\scripts\ai-services\chatgpt\get-ai-suggestions.ps1 -FocusArea all

```

```powershell
# Launch wiki browser
```
