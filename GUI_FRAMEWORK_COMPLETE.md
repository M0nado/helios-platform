## рҹҺЁ GUI FRAMEWORK ARCHITECTURE

### Technology Stack

```
Frontend:
  вң“ WinUI 3 (Modern Windows UI)
  вң“ C# 11
  вң“ XAML for UI design
  вң“ MVVM pattern
  вң“ Fluent Design System

Integration:
  вң“ .NET 7.0 Runtime
  вң“ Windows App SDK
  вң“ Windows Runtime (WinRT)
  вң“ Direct integration with Windows 11

Backend Services:
  вң“ gRPC for service communication
  вң“ SignalR for real-time updates
  вң“ REST APIs
  вң“ Local database (SQLite)

UI Components:
  вң“ Custom controls library
  вң“ Fluent icons (100+ icons)
  вң“ Adaptive layouts
  вң“ Theme system (Dark/Light)
```

```csharp
        _hubConnection = new HubConnectionBuilder()
            .WithUrl("http://localhost:5000/monitoring")
```
