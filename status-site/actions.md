# HELIOS Dashboard Actions

Copy/paste actions for local and hosted control.

## Setup

```bash
./helios.sh setup
```

## Dashboard

```bash
./helios.sh dashboard
```

## Status

```bash
./helios.sh status
```

## Everything safe

```bash
./helios.sh all
```

## GitHub inventory

```bash
./helios.sh github
```

## Azure inventory

```bash
./helios.sh azure
```

## Build graph

```bash
./helios.sh build
```

## Codex tasks

```bash
./helios.sh codex
```

## Recommendations

```bash
./helios.sh recommendations
```

## GitHub hosted workflow

```bash
gh workflow run helios-control-plane.yml -f publish_pages=true -f update_wiki=true
```
