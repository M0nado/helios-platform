# HELIOS Control Center

Static deep-control GUI for setup, agents, merge control, model store, GitHub takeover, Copilot/M365 readiness, Hermes/XCore models, and audit reports. Buttons copy safe shell commands by default.

## Agent Party

The control center renders JRPG-style party cards with fleet XP, levels, abilities, setup-shop commands, model-store commands, and report links. Buttons copy commands by default.
