## Toggle Combinations

Each build supports feature toggling for unlimited combinations:

```powershell
# Start with Standard
.\select-build.ps1 -Variant "standard"

# Add some gaming features
.\toggle-feature.ps1 -Feature "gaming-tools" -Enable $true

# Add AI Hub (from Complete)
.\toggle-feature.ps1 -Feature "ai-hub" -Enable $true

# Remove deep cleaning (keep it light)
.\toggle-feature.ps1 -Feature "deep-clean" -Enable $false

# Result: Custom build combining Standard + Gaming + AI
```

- ✅ Can use Security without Optimization
