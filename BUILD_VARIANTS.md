# HELIOS Build Variants

| Variant | Purpose | Included lanes |
| --- | --- | --- |
| Minimal | Core platform validation | C# platform projects |
| Standard | Application and integration validation | Core, tests, docs, security checks |
| Enterprise | Full production gate | Core, WinUI, native backend, F# analytics, Python AI Hub, Azure deployment dry run |
| Consolidation | External repository intake | Manifest validation, remote reachability, submodule/subtree dry run |
