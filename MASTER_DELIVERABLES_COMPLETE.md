## 📞 TROUBLESHOOTING

### Common Issues

**Problem:** Scripts won't run (execution policy error)
**Solution:** 
```powershell
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser
```

**Problem:** PowerShell emoji rendering issue in dashboard
**Solution:**
```powershell
# Edit ECOSYSTEM-DASHBOARD.ps1, replace emojis with ASCII:
# ✅ → [OK]
# ❌ → [FAIL]
# 🚀 → [EXEC]
```

**Problem:** GitHub CLI not recognized
**Solution:**
```powershell
winget install github.cli
# Then open new PowerShell window
```

**Problem:** NuGet package won't publish
**Solution:**
```bash
# Verify NuGet API key in GitHub Secrets
# Go to: Settings → Secrets → Actions
# Add: NUGET_API_KEY with value from nuget.org
```

**Problem:** GitHub Pages shows 404
**Solution:**
1. Go to repo Settings → Pages
2. Select: Deploy from branch → main → /root
3. Wait 2-3 minutes
4. Refresh browser
