## ❓ FREQUENTLY ASKED QUESTIONS

### Q: Can I use this on macOS or Linux?
**A:** No, HELIOS is Windows-only (Windows 11 Pro). However, you can manage it from any OS via:
- GitHub website (browser)
- GitHub CLI (terminal)
- GitHub Codespaces (web-based)

### Q: How much will this cost?
**A:** Free! HELIOS is open source. Azure/M365 costs vary:
- Individual: $0-50/month
- Team: $50-200/month
- Enterprise: Contact Microsoft for pricing

### Q: Can I stop midway through?
**A:** Yes! Each phase is independent. Stop at Phase 1, 2, or 3. Many users stop at Phase 3 and are perfectly happy.

### Q: What if something breaks?
**A:** Every phase creates snapshots. Rollback with:
```powershell
.\scripts\utilities\rollback.ps1 -SnapshotID "snapshot-name"
```

### Q: Can my team use this?
**A:** Yes! Fork to your organization and:
1. Setup GitHub Projects for tracking
2. Use Codespaces for cloud development
3. Run CI/CD for automated testing
4. Deploy with GitHub Actions

### Q: How do I get support?
**A:** Multiple options:
- 📖 Documentation & guides
- 💬 GitHub Discussions
- 🐛 Report issues
- 📞 Community support
